# Sistema_de_monitero_y_control
El sistema permite medir la temperatura y el nivel de iluminación del entorno, y controlar automáticamente un sistema de ventilación representado por un servomotor y alerta por diodos leds.  Además, el usuario puede activar o desactivar el sistema mediante un pulsador con interrupción, y ajustar la temperatura de referencia usando un potenciómetro.
